# Restaurant Website

A modern, responsive restaurant website built with HTML, CSS (Bootstrap), and JavaScript.

## Features

- Responsive navigation bar
- Image carousel
- About, Team, Gallery, and Contact sections
- Contact form integrated with Telegram
- Interactive map (Leaflet.js)
- Gallery lightbox

## Getting Started

1. Clone the repository:
   ```
   git clone https://github.com/aakashpuri12/Restaurant.git
   ```
2. Open `index.html` in your browser.

## Folder Structure

- `index.html` - Main website
- `styles.css` - Custom styles
- `scripts.js` - Custom scripts
- `img/` - Gallery and team images
- `project/` - Menu and additional images

For any queries, use the contact form on the website.
